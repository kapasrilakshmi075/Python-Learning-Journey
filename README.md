As part of this learning milestone, I created a beginner-level **Student Profile Management System** to apply the Python concepts I learned.

### Concepts Used

- Variables to store student information
- Strings to store and manipulate text data
- Lists to manage skills

The project demonstrates how different Python data types can be combined to organize and manage student profile information.
